# Bellabeat-app-Market-Analysis

# Project Overview

This project analyzes smart device (Fitbit) usage data to uncover user behavior patterns and provide insights to improve marketing strategy for Bellabeat, a health-focused smart device company for women.

The goal is to understand how users interact with fitness trackers and identify opportunities to enhance user engagement and product positioning.

# Business Objective
Analyze smart device usage data
Identify trends in activity, sleep, and calorie burn
Provide actionable insights for Bellabeat’s marketing strategy
# Dataset
Fitbit Fitness Tracker Data (Kaggle)
Multiple datasets used:
Daily Activity
Daily Steps
Sleep Data
Hourly Steps
Hourly Calories
Hourly Intensities
Weight Logs

# Data includes ~30+ users and 22K+ hourly records.

#Tools & Technologies
R
tidyverse, dplyr, ggplot2
Data Cleaning & Transformation
Exploratory Data Analysis (EDA)
# Data Processing
Removed duplicates and handled missing values
Standardized column names
Converted date-time formats
Merged datasets for combined analysis
#  Key Insights
1. Users achieve recommended daily steps (~7500 steps/day)
2. Users do not meet recommended sleep levels (8 hours)
3. Strong positive correlation between steps and calories burned
4. Peak activity observed during evening hours (5 PM – 7 PM)
5. Activity varies across weekdays, with moderate consistency
6. Business Recommendations
Introduce sleep improvement features & alerts
Provide personalized activity reminders during peak hours
Target marketing campaigns based on user activity patterns
Encourage better lifestyle balance (activity vs sleep)
